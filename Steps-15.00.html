KPI Sheet – Health Report vs YOY 15

                                           

1. Open a new sheet by clicking on New Worksheet. Name the sheet as Health Report vs YoY
2. Format the following pills in the following order- 
a. HSR Adopted/Not Adopted - right click - default properties -number format -number (custom) - 2 decimals. 
b. HSR YoY % Difference Adopted/Not Adopted -Right click-default properties-number format-percentage-2 decimals.
3. Drag and drop the State Status on Columns 
4. Drag the following onto the Text Marks card- 
a. Dimension- HSR Category, 
b. HSR Adopted
c. HSR Not Adopted
d. HSR YoY % Difference - Adopted, 
e. HSR YoY % Difference – Not Adopted. 
f. HSR YoY % Difference - Adopted UP, 
g. HSR YoY % Difference - Adopted Down, 
h. HSR YoY % Difference – Not Adopted UP, 
i. HSR YoY % Difference – Not Adopted Down. 
5. Change HSR Adopted and Not Adopted measure to Averages from Sum.
6. Right click on State Status in the viz and select Hide Field Label for columns
7. Format the text on the Text Marks card by clicking on Alignment - horizontal - center - vertical middle. Then click on the ellipses by Text: to open a pop-up window where you will edit the text sizes and colors as follows:
HSR Category   Adopted/Not Adopted -Color  Black:  Font - Tableau Bold : Text size 9  HSR   Adopted    - Color  #00aaff: Font - Tableau Bold: Text size18   
HSR   Not Adopted - Color #2d2463: Font - Tableau Bold: Text size 18
HSR YoY % Difference Adopted -  Color - #ff00ff: Font - Tableau Book: Text size 11 
HSR YoY % Difference Not Adopted-Color - #7719e3: Font- Tableau Book: Text size 11
HSR YoY % Difference UP Adopted/Not Adopted - Color #2b7c40: Font - Tableau Book: Text size 9
HSR YoY % Difference Down Adopted/Not Adopted- Color #ff5500:Font -Tableau Book: Text size 9 
8. Click on Tooltip and uncheck all the boxes
9. Format the worksheet by clicking the Format menu – sheet -  worksheet - none.

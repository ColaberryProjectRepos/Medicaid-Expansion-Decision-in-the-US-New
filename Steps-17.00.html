KPI Sheet – Uninsured 17

                                                


1. Open a new sheet and name it Uninsured Population
2. Format the following pills in the following order-  
a. Uninsured Adopted/Not Adopted - right click - default properties -number format -number (custom) - 2 decimals- Display Unit – Millions(M). 
b. Uninsured % Adopted/Not Adopted -Right click-default properties-number format-percentage-2 decimals. 
3. Drag State status on the columns.
4. Drag the following onto the Text Marks card- 
a. Uninsured Adopted, 
b. Uninsured Not Adopted, 
c. Uninsured % - Adopted, 
d. Uninsured % - Not Adopted, 
e. Uninsured  % - Adopted Up, 
f. Uninsured  % - Adopted Down, 
g. Uninsured  % - Not Adopted UP, 
h. Uninsured  % - Not Adopted Down.
5. Change Uninsured Adopted and Not Adopted measure to Averages from Sum.
6. Right click on State Status in the viz and select Hide Field Label for columns 
7. Format the text on the Text Marks card by clicking on the text and on the Alignment horizontal chose center and vertical chose middle. Then click on the ellipses by Text: to open a pop up window where you will edit the text sizes and colors as follows.

Uninsured  Adopted        Color #00aaff: Font  Tableau Bold: Text size18   
Uninsured  Not Adopted Color  #2d2463: Font  Tableau Bold : Text size 18
Uninsured  %  Adopted                   Color #ff00ff: Font Tableau Book: Text size 11
Uninsured  %  Not Adopted            Color  #7719e3: Tableau Book: Text size 11
Uninsured  %   Adopted/ Not Adopted UP            Color #2b7c40: Font Tableau Book: Text size 9
Uninsured  %  Adopted/Not Adopted Down       Color #ff5500: Font  Tableau Book: Text size 9   
8. Click on Tooltip and uncheck all the boxes.
9. Format the worksheet by clicking the Format menu – sheet - worksheet - none.

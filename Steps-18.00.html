Health Report vs YoY Chart 18

                


1. Open a new worksheet named the sheet Health Report vs YoY.
2. Drag the following unto the viz columns: 
a. Health Category and Year to Rows, 
b. drag HSR Adopted to column 
c. drag HSR Not Adopted to the left axis to create a measure values, 
d.  change the marks to Bar, 
e. drag State Medicaid Status calculated field into color 
f. drag HSR YoY % difference unto tooltip 
g. drag HSR YoY % difference unto rows 
h. change to line and then dual axis, 
i. select measured value in the marks card then remove measured name from under the box.
3. Right Click on HSR Adopted/Not Adopted in the measure value box and change both measures to average.
4. Right click HSR Category/ Year in the viz and select Hide Field Labels for columns.
5. Click on the down arrow on the Measure Names box on the top right of the viz and select edit color. Double click on Avg. HSR-Adopted in the HTML box replace the color with the adopted color #00aaff and click OK. Do the same for Not Adopted with the Not Adopted color #2d2463.  Follow the same steps to edit the State Status colors as well. For Adopted - #ff00ff Not Adopted - #7719e3.
6. Double click on the Value in the left axis to edit the axis. Remove “Value” under the Axis Titles, then close. 
7. Do the same by double clicking on the right axis and remove “HSR YoY % Difference”, close the box.
8. Format the right axis by right clicking on it and selecting format axis. Under axis/scale/numbers, chose percentage and 2 decimal places.
9. Change the view from Standard to Entire view from the top of the viz.
10. Format the worksheet by clicking the Format menu – sheet - worksheet - none.

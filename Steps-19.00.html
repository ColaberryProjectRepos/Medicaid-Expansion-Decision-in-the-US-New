Yearly Uninsured Trend 19

                       


1. Open a new sheet name Yearly Uninsured Trend
2. Drag the following unto the viz
a. Year to columns
b. Uninsured Adopted to Rows,
c. drag Uninsured Not Adopted to the left axis to create a measure values,
d. change the marks to Bar,
e. drag State Medicaid Status calculated field into color
f. drag Uninsured %  unto tooltip
g. drag Uninsured %  unto rows,
h. right click on it and change the Mark Type to line, right click again and select dual axis.
i. select measured value in the marks card then remove measured name from under the box.
3. Right Click on Uninsured Adopted/Not Adopted in the measure value box and change both measures to average.
4. Right click Year in the viz and select Hide Field Labels for columns.
5. Click on the down arrow on the Measure Names box on the top right of the viz and select edit color. Double click on Avg. Uninsured Adopted in the HTML box replace the color with the adopted color #00aaff and click OK. Do the same for Not Adopted with the Not Adopted color #2d2463. Follow the same steps to edit the State Status colors as well. For Adopted - #ff00ff and Not Adopted - #7719e3.
6. Double click on the Value in the left axis to edit the axis. Remove Value under the Axis Titles, then close.
7. Do the same by double clicking on the right axis and remove Uninsured %, then select Logarithmic and symmetric then close the box.
8. Format the left axis by right clicking on it and selecting format axis. Under axis/scale/numbers, chose number(custom), decimal places 2, display units Millions.
9. Format the right axis by right clicking on it and selecting format axis. Under axis/scale/numbers, chose percentage and 2 decimal places.
10. Change the view from Standard to Entire view from the top of the viz.
11.Format the worksheet by clicking the Format menu – sheet - worksheet - none.

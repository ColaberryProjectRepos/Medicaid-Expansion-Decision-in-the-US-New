KPI Sheet – Underlying Cause 16

                                                          


1. Open a new sheet and name it Underlying Cause
2. Format the following pills in the following order- 
a. UC Adopted/Not Adopted - right click - default properties -number format -number (custom) - 2 decimals- Display Unit – Thousand(K). 
b. UC YoY % Difference Adopted/Not Adopted -Right click-default properties-number format-percentage-2 decimals.
3. Drag State status on the columns.
4. Drag the following onto the Text Marks card- 
a. UC Adopted, 
b. UC Not Adopted, 
c. UC YoY % Difference – Adopted, 
d. UC YoY % Difference - Adopted UP, 
e. UC YoY % Difference - Adopted Down, 
f. UC YoY % Difference – Not Adopted, 
g. UC YoY % Difference – Not Adopted UP, 
h. UC YoY % Difference – Not Adopted Down.
5. Click and change UC Adopted/Not Adopted  measure from Sum to Averages
6. Right click on State Status in the viz and select Hide Field Label for columns
7. Format the text on the Text Marks card by clicking on the text and on the Alignment horizontal chose center and vertical chose middle. Then click on the ellipses by Text: to open up a window where you will edit the text sizes and colors as follows.
UC -  Adopted            Color #00aaff: Font -Tableau Bold: Text size18   
UC - Not Adopted      Color #2d2463: Font -Tableau Bold: Text size 18
UC YoY % Difference     Adopted            Color  #ff00ff: Tableau Book: Text size 11
UC YoY % Difference     Not Adopted    Color   #7719e3: Tableau Book : Text size 11
UC YoY % Difference UP Adopted/Not Adopted    Color #2b7c40: Tableau Book: Text size 9
UC YoY % Difference Down  Adopted/Not Adopted  Color #ff5500: Tableau Book: Text size 9   
8. Click on Tooltip and uncheck all the boxes.

9. Format the worksheet by clicking the Format menu – sheet - worksheet - none.

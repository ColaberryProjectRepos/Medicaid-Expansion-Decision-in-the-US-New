Underlying Cause of Death vs YoY 20

                     


1. Open a new sheet name Underlying Cause of Death vs YoY.
2. Drag the following unto the viz
a. Year to columns
b. UC Adopted to Rows,
c. drag UC Not Adopted to the left axis to create a measure values,
d. change the marks to Bar,
e. drag State Medicaid Status calculated field into color
f. drag UC YoY % Difference  unto tooltip
g. drag UC YoY % Difference unto rows,
h. right click on it and change the Mark Type to line, right click again and select dual axis.
i. select measured value in the marks card then remove measured name from under the box.
3. Right Click on UC Adopted/Not Adopted in the measure value box and change both measures to average.
4. Right click on Year in the viz and select Hide Field Labels for columns.
5. Click on the down arrow on the State Medicaid Status box on the top right of the viz and select edit color. Double click on Avg. UC-Adopted in the HTML box replace the color with the adopted color #00aaff and click OK. Do the same for Not Adopted with the Not Adopted color #2d2463. Follow the same steps to edit the State Status colors as well. For Adopted - #ff00ff Not Adopted - #7719e3.
6. Double click on the Value in the left axis to edit the axis. Remove Value under the Axis Titles, then close.
7. Do the same by double clicking on the right axis and remove UC YoY % Difference, then select Logarithmic and symmetric then close the box.
8. Format the left axis by right clicking on it and selecting format axis. Under axis/scale/numbers, chose number(custom), decimal places 2, display units Thousands.
9. Format the right axis by right clicking on it and selecting format axis. Under axis/scale/numbers, chose percentage and 2 decimal places.
10. Change the view from Standard to Entire view from the top of the viz.
11. Format the worksheet by clicking the Format menu – sheet - worksheet - none.
